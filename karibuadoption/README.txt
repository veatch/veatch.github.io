This is an archive of Sheila Veatch's website, karibuadoption.me.
It was archived by the Wayback Machine (archive.org).
This copy was downloaded with the wayback-machine-downloader tool
(https://github.com/hartator/wayback-machine-downloader) in March
2022 by Sheila's nephew, Jason Veatch.

index.html shows the site's state from September 2017. The next
snapshot on archive.org is from 2019, when Sheila no longer had
the domain. Unfortunately, images and CSS (page styles) were not
included in the archive. If you browse into the year directories,
you can see the site's posts going back to 2013.
